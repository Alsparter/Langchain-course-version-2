🦜🔗 LangChain Course — Version 2

A structured, hands-on course covering ~90% of LangChain's core concepts through Jupyter Notebooks. This version does not include Agents, but gives you a rock-solid foundation in everything else LangChain has to offer.

🚀 Getting Started
1. Clone the repository
bashgit clone https://github.com/Alsparter/Langchain-course-version-2.git
cd Langchain-course-version-2
2. Install dependencies
bashpip install -r requirement.txt
3. Set up your API Key
Create a .env file in the root directory and add your OpenAI API key:
envOPENAI_API_KEY=your_api_key_here
4. Launch Jupyter Notebook
bashjupyter notebook
Then open the notebooks in order starting from chapter1.ipynb.

🛠️ Prerequisites

Python 3.8 or higher
Basic Python programming knowledge
An OpenAI API key (or compatible LLM provider)
Jupyter Notebook or JupyterLab installed


🧠 What You'll Learn

✅ LangChain fundamentals — LLMs, Prompts, and Chains
✅ Memory and conversation history management
✅ Loading and splitting documents
✅ Vector stores and embeddings
✅ Retrieval-Augmented Generation (RAG)
✅ Output parsers and structured outputs
✅ Building real-world LangChain pipelines
❌ Agents (not covered in this version)


⚠️ Notes

Run the chapters in order — each one builds on the previous
This is Version 2 of the course, updated for the latest LangChain APIs
Make sure your API key is set before running any notebook that calls an LLM
